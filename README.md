# NetPilot Community

This repo is the central place to report bugs, request features, and discuss anything related to [NetPilot](https://netpilot.io) — the AI-powered network lab platform.

## How to participate

- **Bug reports** — [Open an issue](https://github.com/netpilot-labs/community/issues/new?template=bug_report.md)
- **Feature requests** — [Open an issue](https://github.com/netpilot-labs/community/issues/new?template=feature_request.md)
- **Questions & discussion** — [Start a discussion](https://github.com/netpilot-labs/community/discussions)
